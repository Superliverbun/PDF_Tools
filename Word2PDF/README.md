# Word 批次轉 PDF 工具

一個簡單的工具，可以把整個資料夾裡的 Word 檔案全部轉換成 PDF。

## 📁 使用方法

1. **執行程式**：
   ```bash
   python main.py
   ```

2. **選擇資料夾**：
   - 選擇包含 Word 檔案的資料夾
   - 選擇要輸出 PDF 的資料夾

3. **開始轉換**：
   - 點擊「開始轉換」按鈕
   - 等待轉換完成

## 🔧 需要安裝

- Python 3.6+
- Microsoft Word
- 安裝套件：
  ```bash
  pip install pywin32
  ```

## 📋 檔案說明

- `main.py` - 有圖形介面的主程式
- `nogui.py` - 純指令版本（修改程式碼裡的路徑就能用）

## 📂 轉換範例

```
輸入資料夾/
├── 報告.docx
└── 子資料夾/
    └── 文件.docx

↓ 轉換後

輸出資料夾/
├── 報告.pdf
└── 子資料夾/
    └── 文件.pdf
```

就是這麼簡單！一鍵把所有 Word 檔變成 PDF。
